New kid on the Blockchain.

A huge barrier for lay users is the number of cryptocurrencies. While exchanges solve the problem for advanced users, there remains no easy way move from one cryptocurrency to another. Until now. Using Raft+DPoS algorithms with alliance nodes, Cryptoworld will offer automatic transfers between blockchain wallets, including Bitcoin and Ethereum.

Beyond the concrete.

Cryptoworldʼs virtual assets are represented by real-world buildings. Whether landmarks like the Empire State building or a small town library, they function similarly in Cyrptoworld, allowing owners to profit as users purchase services.

A place called home.

For every 1,000 CWVs owned, users enter a lottery that raffles a property randomly. Using GoogleMapʼs API interface, a randomly selected longitude and latitude will come into possession of the user, ownership guaranteed through Cryptoworldʼs blockchain.
